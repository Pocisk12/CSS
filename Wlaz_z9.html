  4. Ile arkuszy styli może być dołączanych do jednego dokumentu HTML oraz jaka zasad musi być zachowana przy większej ilości dołączanych styli?

    Do jednego dokumentu HTML można podłączyć kilka plików CSS. W przypadku większej liczby arkuszy ważne jest, w jakiej kolejności są one ładowane – style z plików dodanych później mogą zastąpić reguły z wcześniejszych arkuszy.

  5. Co się dzieje w przypadku konfliktów?

    Gdy ten sam styl jest zdefiniowany w kilku plikach CSS, przeglądarka wybierze ten z arkusza, który został załadowany jako ostatni. Można też używać bardziej precyzyjnych selektorów albo dodać !important, by wymusić zastosowanie konkretnej reguły.

  6. Jak są tworzone i edytowane pliki arkuszy z użyciem edytora tekstowego?

    Plik CSS można stworzyć i edytować nawet w prostych programach, takich jak Notatnik czy Vim. Ważne, by zapisać go z rozszerzeniem .css.

  7. Jak są tworzone i edytowane pliki arkuszy z użyciem specjalnego edytora CSS?

    Profesjonalne edytory, takie jak Visual Studio Code, Sublime Text czy Brackets, oferują funkcje ułatwiające pracę – m.in. podświetlanie składni, autouzupełnianie kodu oraz podgląd efektów na żywo.
